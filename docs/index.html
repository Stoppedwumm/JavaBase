Getting Started

Erste Schritte

JavaBase is a minimal framework wrapping Swing and AWT to create a game loop and provide simple drawing commands. It combines standard canvas rendering with "Turtle Graphics" commands (move, pen down, etc.).

To start a new project, clone the template:

JavaBase ist ein minimales Framework, das Swing und AWT kapselt, um einen Game-Loop zu erstellen und einfache Zeichenbefehle bereitzustellen. Es kombiniert Standard-Canvas-Rendering mit "Turtle-Grafik"-Befehlen (Bewegen, Stift runter, etc.).

Um ein neues Projekt zu starten, klonen Sie das Template:

git clone https://github.com/Stoppedwumm/JavaBaseTemplate

Modify the entry point at: Bearbeiten Sie den Einstiegspunkt unter: app/src/main/java/example/project/NeonDodger.java

package engine.core

public abstract class CoreGame

The heart of the engine. Your game class must extend this.

Das Herz der Engine. Ihre Spielklasse muss von dieser Klasse erben.

package engine.loader

public class Init

The main entry point (`public static void main`). It uses Reflection to scan the classpath for any class annotated with @Game that extends CoreGame.

This allows the engine to be decoupled from your specific game code.

Der Haupteinstiegspunkt (`public static void main`). Es nutzt Reflection, um den Classpath nach Klassen zu scannen, die mit @Game annotiert sind und CoreGame erweitern.

Dies erlaubt eine Entkopplung der Engine von Ihrem spezifischen Spielcode.

// engine/annot/Game.java
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
public @interface Game {
    String name() default "Unknown Game";
}

Utility: Loading Images

Utility: Bilder laden

package my.project.utils; 

import javax.imageio.ImageIO;
import java.awt.Color;
import java.awt.image.BufferedImage;
import java.io.IOException;
import java.io.InputStream;

public class Image {
    /**
     * Loads an image from resources and returns a Color array.
     */
    public static Color[][] loadPixelMap(String resourcePath) throws IOException {
        InputStream is = Image.class.getResourceAsStream(resourcePath);
        if (is == null) throw new IOException("Resource not found: " + resourcePath);

        BufferedImage image = ImageIO.read(is);
        int width = image.getWidth();
        int height = image.getHeight();
        Color[][] pixelMap = new Color[width][height];

        for (int x = 0; x < width; x++) {
            for (int y = 0; y < height; y++) {
                pixelMap[x][y] = new Color(image.getRGB(x, y), true);
            }
        }
        return pixelMap;
    }
}
